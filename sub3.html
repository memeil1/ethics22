- 발표주제: 발표주제: 교원성과상여금/교원능력개발평가
과제 상세 : 주제 논술 레포트
- 주제: 21세기 교사의 교육리더십

- 자신의 주장을 위주로 진술하되 주장을 논리적으로 뒷받침 할 이론, 원리 등을 반드시 적용 하여 작성해야 한다. 자신만의 지적고민과 독서를 통해 작성하지 않고 인터넷 등에서 수집 한 자료들을 짜깁기한 글은 평가대상에서 제외되고 수강생은 재수강대상으로 분류된다.
- 하나의 완성된 글(구성체계를 갖출 것)로 작성한다.
- 자필로 작성한다.(임용시험1차 교직논술대비 연습 목적)
- 제출기일(12주차) 엄수(제출기간 초과시 감점10점)한다.
- 분량은 A4용지 2쪽(양식 참조) 이내로 작성한다.
과제 상세 : 교육 제도 공모전
조별 활동(발표조) : 21세기 새로운 교육제도 아이디어 공모전

- 내용 : 현행 제도의 특성과 문제점 분석 / 새로운 교육제도 모형 제시(인사, 조직, 재정, 법령 포함) / 제안된 새로운 제도 도입 시 예 상되는 변화
- 형식 : PPT
- 발표시간 : 15분(질의응답 5분)
